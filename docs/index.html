In our daily life, we have so many idées to write down, and many of us are suffering from the work of complicated text formatting. We usually save our data on many cloud file storage service, such as Dropbox, Google drive, 百度云, and so on. But what has happened did prove that these cloud storage is not reliable.
Makrdown is a lightweight markup language which is a great way to save us from it.
Idees is a markdown editor and support these self deployed file storage services.
